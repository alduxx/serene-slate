# Código Como Poesia

Existe uma estética no código bem escrito que transcende sua funcionalidade. É a diferença entre uma solução que funciona e uma solução que canta.

## A Arte da Simplicidade

Código elegante é como um poema haiku - cada palavra conta, cada espaço tem propósito. Não é sobre fazer menos por preguiça, mas sobre fazer o essencial com maestria.

```javascript
// Isto funciona, mas não canta
function calc(a,b,c){return a>b?a>c?a:c:b>c?b:c}

// Isto respira
function findMaximum(first, second, third) {
  return Math.max(first, second, third)
}
```

## Legado em Linhas

O código que escrevemos hoje será lido por alguém amanhã - talvez por nós mesmos. Que história queremos contar?

Cada função é um verso, cada módulo é uma estrofe, cada aplicação é um poema completo. Escreva com intenção.
