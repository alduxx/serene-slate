# Horizonte

O horizonte recua  
na mesma velocidade  
que avanço

Promessa perpétua  
de chegada  
que nunca se cumpre

Mas aprendi:  
não caminhamos  
para alcançar o horizonte

Caminhamos  
para ver a paisagem  
mudar

E quando finalmente olho  
para trás  
percebo:

O que procurava à frente  
estava sempre  
nos passos dados
